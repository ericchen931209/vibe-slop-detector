## 7. Interpretation

The AUC of **0.4943** is near random chance — static patterns alone may not reliably discriminate.

- AI-assisted files score **-1.8 points higher** on average (59.75 vs 61.51).
- The most discriminative categories are those with the highest AI/Human ratios in Section 4.
- Statistical test confirms the score difference is not statistically significant at α=0.05.
- Effect size r = 0.4943 (medium (r > 0.3)) suggests a real and notable practical difference.

### Limitations

- **Label noise:** "AI-assisted" label is inferred from README mentions — some repos may be human-written despite mentions, or partially AI-generated.
- **Static analysis only:** LLM-detected categories (S1 Ghost Comment, S10 Verbosity Inflation, S11 Redundant Docstring) were not used.
- **Language:** Python only (v0.1 supports Python exclusively).
- **Sample bias:** Human repos selected by star count — popular code may be cleaner than average.
